# Spieletag System

Eine Webanwendung zur einfachen und unterhaltsamen Verwaltung von Spieleabenden, Turnieren oder Spieletagen. Dieses Tool nimmt dir die Organisation ab, mischt Spiele zufällig, verwaltet Spielerpunkte und bietet spezielle Events wie Dancebreaks!

## 🌟 Funktionen

- **Spielerverwaltung:** Einfaches Hinzufügen und Entfernen von Spielern.
- **Drei Spiele-Kategorien:** 
  - 🎮 **Hauptspiele:** Progressive Punktvergabe für längere/wichtige Spiele.
  - 🎲 **Zwischenspiele:** Individuelle Punkte für schnelle Runden zwischendurch.
  - ⚠️ **Strafspiele:** Spezielle Spiele mit eigenen Punktwerten.
- **Zufalls-Ziehung (Mischen):** Spiele werden dynamisch, basierend auf einstellbaren Wahrscheinlichkeiten (Urnen-Prinzip oder feste Chancen), gezogen.
- **Event-System (Dancebreak):** Überraschende Tanzpausen mit einstellbarer Musiklänge und Chance.
- **Live-Scoreboard:** Behalte die Punkte aller Teilnehmer jederzeit im Blick.
- **Ende & Gewinner:** Konfigurierbare Endbedingungen (z.B. wenn alle Hauptspiele gespielt wurden) und animierte Siegerehrung.
- **Dark Mode:** Angenehm für die Augen in dunklen Umgebungen.

## 📱 Einrichten und Hostbar über GitHub Pages (Ideal für iPads/Tablets)

Um das Spieletag System über dein iPad (oder jedes andere Gerät) abzurufen, ist das Hosting über GitHub Pages der einfachste und kostenlose Weg. So richtest du es ein:

### Schritt 1: GitHub Repository erstellen
1. Erstelle einen (kostenlosen) Account auf [GitHub.com](https://github.com/), falls du noch keinen hast.
2. Klicke oben rechts auf das **`+`** und wähle **`New repository`**.
3. Gib dem Repository einen passenden Namen (z.B. `spieletag`).
4. Setze es auf **Public** (damit GitHub Pages funktioniert).
5. Klicke auf **Create repository**.

### Schritt 2: Dateien hochladen
1. Klicke auf der neuen Repository-Seite auf **`uploading an existing file`**.
2. Ziehe einfach die Datei `SpieletagSystem.html` sowie diese `README.md` Datei in das Fenster.
3. Wichtig: Nenne die `SpieletagSystem.html` am besten in `index.html` um, dann ist der Link kürzer!
4. Klicke auf **Commit changes**.

### Schritt 3: GitHub Pages aktivieren
1. Gehe in deinem Repository oben auf **`Settings`** (Zahnrad-Symbol).
2. Klicke auf der linken Seite im Menü auf **`Pages`**.
3. Wähle unter `Source` den Branch **`main`** (oder `master`) aus dem Dropdown und belasse den Ordner auf `/(root)`.
4. Klicke auf **Save**.
5. Nach ca. 1-2 Minuten zeigt GitHub oben einen Link an (z.B. `https://dein-username.github.io/spieletag/`). 

Dieser Link ist ab jetzt deine persönliche URL für das Spieletag System! Speichere ihn als Favorit auf deinem iPad oder lege ihn als App-Icon auf dem Homescreen ab.

## 🕹️ Nutzung
1. Öffne den Link über dein Tablet, Handy oder PC.
2. Trage eure Namen ein.
3. Setze die Einstellungen (Wahrscheinlichkeiten, Punkte etc.).
4. Füge eure Spielebecken mit den "+ Spiel" Buttons hinzu.
5. Klicke auf "Spieletag starten" und zieht abwechselnd ein Spiel!
